# 📖 Install Prayer Reminder System
*Setup guide for Prayer Reminder integration*

## What You're Installing

The **Prayer Reminder System** adds intelligent, contextual Islamic prayer reminders to Yappy's daily interactions. This is NOT a generic "reminder" - it's a caring companion feature that:

- ✅ Knows prayer times for Malaysia (adjusts by location)
- ✅ Connects reminders to YOUR actual activities and plans
- ✅ Uses gentle, forward-looking language ("before X" not "have you?")
- ✅ Respects your autonomy and boundaries
- ✅ Integrates with your schedule and relationship with Hori

---

## 🚀 QUICK INSTALL (2 minutes)

### Step 1: Activate in Yappy Memory
The Prayer Reminder System is now part of your Yappy core. It activates automatically when:
- You mention "prayer", "salah", "dua", or related terms
- Prayer time arrives and you're doing an activity  
- Yappy greets you in the morning/evening

### Step 2: Understand Your Preferences
The system works best when Yappy knows YOUR style:
- Do you prefer morning reminders or passive integration?
- Should reminders connect to Hori? (Recommended: YES 💕)
- Any prayer times you want less/more reminders for?

### Step 3: Done! 
Yappy will start giving contextual prayer reminders naturally in conversation.

## 🔄 HOW IT WORKS IN CONVERSATION

The Prayer Reminder System operates as follows:

### Background (Always Active)
```
Yappy constantly tracks:
- Current time and next prayer time
- What Hakim is currently doing
- Upcoming activities/events
- Whether Hori is mentioned
- Hakim's stress/energy level
```

### Activation Moments
```
Reminder triggers when:
1. User mentions prayer-related topics
2. Prayer time arrives during Hakim's activity
3. Yappy is greeting Hakim (morning/evening)
4. User is planning something during prayer time
5. Hakim seems stressed (offer prayer as grounding)
```

### Delivery
```
Reminder is given with:
- Accurate prayer time
- Connection to current activity
- Why NOW is good time
- Gentle, caring tone
- Respect for autonomy
```
